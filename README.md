# 📊 Predictive Analytics – Excel Forecasting Models

**Tools:** Microsoft Excel | Pivot Tables | Solver | Forecasting Functions

## 📌 Project Overview
Built Excel-based predictive analytics models using historical data 
to support business planning, trend analysis and risk assessment.

## 🔍 Techniques Used
- Time series forecasting
- Trend analysis and moving averages
- Data validation and accuracy checks
- Pivot Tables for data summarization
- Solver for optimization modelling
